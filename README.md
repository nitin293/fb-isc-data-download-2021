# Uses:

## For Linux based environment

Commands : 

```
chmod 777 run.sh
bash run.sh
python3 download.py
```

## For windows environment

### Requirements :

You must have install `python3` and `pip` in your P.C.

If you have `python3` and `pip` installed in your P.C.

Command : 

```
pip install -r requirements.txt
python download.py
```
